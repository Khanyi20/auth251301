<h2>Creative Chess Authentication System(MERN Stack) </h2>

<h2>Description</h2>
<p>A creative MERN Stack authentication system that combines traditional password authentication with a chess-piece pattern verification system.

Users register using:
- Email
- Password
- Custom chess-piece pattern

During login, users must:
1. Enter correct email and password
2. Recreate their saved chess-piece pattern</p>
